### 3.2 操作记录表（api_operationrecord）

#### 3.2.1 表结构说明

表名：`api_operationrecord`

用途：用于保存从小程序端或网页端传输过来的每一次报到操作痕迹，作为验证数据链路通畅的直接证据。

#### 3.2.2 字段详细说明

| 字段名 | 数据类型 | 是否可为空 | 默认值 | 主键 | 外键 | 说明 |
|--------|----------|------------|--------|------|------|------|
| id | IntegerField | NO | 自动递增 | ✅ | - | 自增主键，唯一标识每条记录 |
| remark | CharField(200) | YES | '新生报到操作' | - | - | 备注信息，用于简单说明数据来源 |
| created_at | DateTimeField | NO | 自动记录 | - | - | 创建时间，在记录生成时自动填入当前时间 |

#### 3.2.3 字段详细定义

**id 字段**
```python
id = models.AutoField(primary_key=True)
```
- 数据类型：整数型
- 主键：是
- 自增：是
- 说明：系统自动生成的唯一标识符

**remark 字段**
```python
remark = models.CharField(max_length=200, default='新生报到操作', verbose_name='备注信息')
```
- 数据类型：可变长度字符串
- 最大长度：200 字符
- 默认值：'新生报到操作'
- 中文标签：备注信息
- 说明：用于简单说明数据来源，可自定义

**created_at 字段**
```python
created_at = models.DateTimeField(auto_now_add=True, verbose_name='创建时间')
```
- 数据类型：日期时间型
- 自动填充：是（创建时自动记录）
- 中文标签：创建时间
- 说明：在记录生成时自动填入当前时间，精确到毫秒

#### 3.2.4 索引设计

| 索引名 | 索引类型 | 索引字段 | 说明 |
|--------|----------|----------|------|
| PRIMARY | 主键索引 | id | 自动创建，保证唯一性 |
| api_operationrecord_created_at | 普通索引 | created_at | 用于按时间排序查询 |

#### 3.2.5 表约束

| 约束类型 | 约束说明 |
|----------|----------|
| 主键约束 | id 字段为主键，确保唯一性 |
| 非空约束 | id、created_at 字段非空 |
| 默认值约束 | remark 字段默认为 '新生报到操作' |
| 自动时间约束 | created_at 字段在插入时自动记录当前时间 |

## 5. SQL 语句

### 5.1 建表语句

```sql
CREATE TABLE api_operationrecord (
    id INTEGER NOT NULL PRIMARY KEY AUTOINCREMENT,
    remark VARCHAR(200) NOT NULL DEFAULT '新生报到操作',
    created_at DATETIME NOT NULL
);

CREATE INDEX api_operationrecord_created_at ON api_operationrecord (created_at);
```

### 5.2 插入数据

```sql
INSERT INTO api_operationrecord (remark, created_at) 
VALUES ('新生报到操作', '2026-04-24 10:30:00.000000');
```

### 5.3 查询数据

```sql
-- 查询所有记录（按时间倒序）
SELECT * FROM api_operationrecord ORDER BY created_at DESC;

-- 查询最新一条记录
SELECT * FROM api_operationrecord ORDER BY created_at DESC LIMIT 1;

-- 统计报到总人数
SELECT COUNT(*) FROM api_operationrecord;
```

### 5.4 更新数据

```sql
-- 更新备注信息
UPDATE api_operationrecord SET remark = '自定义备注' WHERE id = 1;
```

### 5.5 删除数据

```sql
-- 删除指定记录
DELETE FROM api_operationrecord WHERE id = 1;

-- 清空所有记录
DELETE FROM api_operationrecord;
```

## 6. Django 模型定义

### 6.1 模型代码

```python
from django.db import models
from django.utils import timezone


class OperationRecord(models.Model):
    """操作记录表 - 用于保存从小程序端传输过来的操作痕迹"""
    
    remark = models.CharField(
        max_length=200,
        default='新生报到操作',
        verbose_name='备注信息'
    )
    created_at = models.DateTimeField(
        auto_now_add=True,
        verbose_name='创建时间'
    )

    class Meta:
        verbose_name = '操作记录'
        verbose_name_plural = '操作记录'
        ordering = ['-created_at']

    def __str__(self):
        return f"{self.id} - {self.remark} - {self.created_at}"
```

### 6.2 模型说明

| 属性 | 说明 |
|------|------|
| verbose_name | 单数形式的中文名称 |
| verbose_name_plural | 复数形式的中文名称 |
| ordering | 默认排序方式（按创建时间倒序） |
| __str__ | 对象的字符串表示 |
