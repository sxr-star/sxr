# 新生报到系统 - 使用说明书

## 1. 系统简介

新生报到系统是一套用于高校新生报到场景的信息化管理工具。该系统实现了从前端网页/小程序到后端数据库的完整数据链路，可以验证前后端联调是否正常工作。

### 1.1 系统架构

```
┌─────────────────┐         ┌─────────────────┐         ┌─────────────────┐
│   网页版/小程序   │ ──────▶│   Django 后端   │ ──────▶│   SQLite 数据库  │
│   (前端界面)    │  HTTP   │   (API 接口)    │  SQL   │   (数据存储)    │
└─────────────────┘         └─────────────────┘         └─────────────────┘
```

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | HTML + CSS + JavaScript（网页版）、微信小程序 |
| 后端 | Django 6.0.4 |
| 数据库 | SQLite 3 |
| 开发环境 | Python 3.10+ |

## 2. 环境要求

### 2.1 软件要求

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Python | 3.10 或更高 | 运行环境 |
| pip | 最新版本 | Python 包管理工具 |

### 2.2 Python 环境检查

打开命令提示符（PowerShell），运行以下命令检查 Python 是否已安装：

```powershell
python --version
```

如果显示版本号（如 `Python 3.14.0`），则表示 Python 已安装。

## 3. 安装步骤

### 3.1 方式一：快速安装（推荐）

如果你是首次运行该项目，请使用 `完整安装启动.bat`：

1. 双击运行 **完整安装启动.bat**
2. 等待自动完成以下操作：
   - 安装 Django 框架
   - 安装必要的依赖包
   - 初始化数据库
   - 启动服务器
3. 启动成功后，浏览器会自动打开

### 3.2 方式二：手动安装

如果自动安装失败，可以手动执行以下步骤：

#### 步骤 1：安装 Django

```powershell
pip install django
```

#### 步骤 2：安装依赖

```powershell
pip install django-cors-headers
```

#### 步骤 3：初始化数据库

```powershell
python manage.py migrate
```

#### 步骤 4：创建管理员账户（可选）

```powershell
python manage.py createsuperuser
```

#### 步骤 5：启动服务器

```powershell
python manage.py runserver
```

## 4. 运行项目

### 4.1 启动服务器

双击运行 **启动项目.bat**，或者在命令行中运行：

```powershell
python manage.py runserver
```

启动成功后，会显示类似以下信息：

```
Watching for file changes with statreloader...
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

### 4.2 访问系统

启动成功后，打开浏览器访问：

| 功能 | 地址 |
|------|------|
| 网页版报到 | http://127.0.0.1:8000/ |
| 管理后台 | http://127.0.0.1:8000/admin/ |

### 4.3 快速启动

如果已完成首次安装，可以直接双击 **快速启动.bat** 一键启动项目。

## 5. 功能使用

### 5.1 网页版报到

1. 打开浏览器访问 `http://127.0.0.1:8000/`
2. 点击页面上的 **「报到」** 按钮
3. 系统会显示 "报到成功" 提示
4. 数据会自动保存到数据库

### 5.2 微信小程序报到

1. 打开微信开发者工具
2. 导入 `小程序` 文件夹
3. 点击页面上的 **「报到」** 按钮
4. 同样可以完成报到操作

### 5.3 管理后台

1. 打开浏览器访问 `http://127.0.0.1:8000/admin/`
2. 使用管理员账户登录
3. 可以查看所有报到记录

---

## 6. 数据查看

### 6.1 通过管理后台查看

1. 访问 `http://127.0.0.1:8000/admin/`
2. 登录后点击 **「操作记录」**
3. 可以看到所有报到记录

### 6.2 直接查看数据库

数据库文件位于：`新生报到系统\db.sqlite3`

可以使用 SQLite 工具打开查看，或在 Django shell 中查询：

```powershell
python manage.py shell
```

```python
from api.models import OperationRecord
records = OperationRecord.objects.all()
for r in records:
    print(r.id, r.remark, r.created_at)
```

## 7. 常见问题

### 7.1 端口被占用

如果启动时提示端口被占用，可以更换端口：

```powershell
python manage.py runserver 8080
```

### 7.2 数据库错误

如果数据库出现问题，可以重新初始化：

```powershell
python manage.py migrate
```

### 7.3 网页无法访问

1. 确认服务器已启动
2. 检查防火墙设置
3. 确认访问地址是否正确
